1. About VS Code:

Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality.



2. Installing VS Code:

Firstly on your chrome browser type 'vs code download' and then click on first link that appears on the result screen. After clicking, click on the windows option and it will start to download the vs code executer. After downloading, click on the file and open it and now you have to do some setups.



3. Setuping the VS Code

After clicking on the downloaded file, a sub-screen will appear. In the first screen choose on the radio button that have the following message,"I accept the agreement" and then click on next. After that again a screen will appear. Now, in this screen, select all the check boxes and click on next. After that again a screen will appear. Finally in this screen click on install and your VS Code IDE will start downloading and your VS Code setup is complete..



4. About Console

A console application is a program designed to be used via a text-only computer interface, such as a text terminal, the command line interface of some operating systems (Unix, DOS, etc.) or the text-based interface included with most Graphical User Interface (GUI) operating systems, such as the Windows Console in Microsoft Windows, the Terminal in macOS, and xterm in Unix.



5. Uses of Console

The Console can be used to log information as part of the JavaScript development process, as well as allow you to interact with a web page by carrying out JavaScript expressions within the page's context. Essentially, the Console provides you with the ability to write, manage, and monitor JavaScript on demand.



6. Printing Message on Console

To print a message in console, open your javascript file and type a function "console.log()" and type the value inside the parenthesis to print it. If it is a string value, then put a double quotation and then write the value. If it is a integer value, then just write the number inside the round brackets. If it is a float number, then also apply the same method like for integers. If you want to print a variable then declare a variable first and then declare a value inside it and then to print the value inside the varible, just write the name of the variable and the value inside it will be printed in the console.



7. Opening The Console

To ope the console, right click on the page which you want to inspect. A submenu will appear. There click on inspect option. Then a side menu will appear. Then, click on the console option on the top right menu. After that your console will be opened.
